# YOLO Streamlit Dashboard
Display predicted Video, Images and webcam using YOLO models (**YOLOv7**) with Streamlit

## Streamlit Options
### Modes
 - RTSP
 - Webcam
 - Video
 - Image
 

 ## StepUp
```
git clone
cd streamlit-yolo
```
Install dependency
```
pip3 install -r requirements.txt
```
Run **Streamlit**
```
streamlit run app.py
```
